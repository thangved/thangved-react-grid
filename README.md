# thangved-react-grid

## Install

```bash
npm install --save thangved-react-grid
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'thangved-react-grid'
import 'thangved-react-grid/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
